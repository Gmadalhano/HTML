Imagens Dinâmicas

Imagens podem ser acessadas de vários dispositivos móveis diferentes, com tamanhos de telas diferentes. As imagens do site devem se adaptar ao dispositivo utilizado, visando a otimização de resposta e dados do site.

Através da tag <picture> têm-se acesso ao recurso <source:media:type> onde é possível inseririmagens de tamanhos diferentes para telas diferentes, por meio dos pixels.

O recurso <source:media:type> deve ser aplicado de forma crescente, a imagem menor primeiro, e depois as maiores. A imagem original deve ser o ultimo elemento do comando <picture>

Reproduzindo Áudios

Vamos aprender a reproduzir áudios.

Basta usar o comando <audio> e adicionar os parâmetros controls para inserir o controlador

Fazendo assim, direto, não há a possíbilidade de escolher o formato ou tipo de arquivo (mp3, ogg, wave), e pode ser que o navegador não tenha compatibilidade com o typo proposto.

Existe uma outra forma de disponibilizar áudios para reprodução de maneira parecida, onde é possível disponibilizar o mesmo áudio em diferentes formatos para maior taxa de compatibilidade. A diferença é que deve-se mudar o "tipo de source. Apaga-se o source sugerido pelo comando audio, e digita novamente, selecionando o source:src. Dessa maneira será necessário adicionar o "type" que podem ser vários como audio/mpeg, audio/wave, audio/ogg

Caso seu navegador não reconheça o arquivo clique aqui para baixar o arquivo.

Existe para o segundo caso o parametro loop que reinicia o áudio após o termino.
