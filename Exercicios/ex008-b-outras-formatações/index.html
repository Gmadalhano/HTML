Código Fonte - Pré Formatado

O comando document.getElement("teste) é escrito em linguágem JavaScript

A formatação do trecho em uma fonte monoespácial, isto é, com todos os caracteres ocupando o mesmo espaço na linha, é possível com a tag <code>, utilizado para indicar e facilitar a leitura de códigos.

Porém, a tag sozinha não garante a formatação do código, de maneira hierarquicamente falando, o texto perde identação.

Para mater a estrutura de um código, ou de um texto nas condições de um código, usamos a tag <pre>.

A seguir veremos o mesmo código com e sem a tag "pre", o primeiro sem a aplicação da tag, e segundo utilizando a tag para garantir a formatação do código.

Sem a tag <pre>

number = int(input('Digite um número inteiro: ')) if number % 2 == 0: print(f'O valor {number} é PAR') else: print(f'O valor {number} é IMPAR') print('Fim do programa')

Com a tag <pre>

        
            number = int(input('Digite um número inteiro: '))
            if number % 2 == 0:
                print(f'O valor {number} é PAR')
            else:
                print(f'O valor {number} é IMPAR')
            print('Fim do programa')

As aspas inseridas de forma manual são utilizadas para dar sentido no texto, destacar palavras estrangeiras ou ironia. Exemplo:

Siglas e Abreviações

Quando siglas aparecem no texto, pode-se declarar o significado das letras através da tag <abbr>, que disponibiliza um pop-up com o significado completo de uma sigla ou abreviação. Ao chamar a tag, ela automaticamente solicita um "titulo" que é significado da sigla ou abreviação. Por exemplo:

Estamos estudando HTML5 e CSS3 e esse é um teste de siglas.

As siglas ficaram sublinhadas e ao posicionar o cursor do mause sobre as siglas o conteudo dos pop-up aparecem na tela.
