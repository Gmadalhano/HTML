Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito utilizando a tag <b>.
A tag em questão apenas altera a forma do conteúdo, o que não é preocupação da linguagem HTML5.

Nesta frase temos um termo em destaque utilizando a tag <strong>
Dessa maneira, além de alterar a forma do texto é dado uma maior relevancia semantica a informação.

Itálico / Ênfase

Nesta frase temos um termo em italico utilizando a tag <i>. (não semantica)

Nesta frase temos um termo com ênfase utilizando a tag <em>. (Aplicação semântica)

Textos Marcados

Podemos criar paragrafos com trechos destacados por marcação colorida

Textos Grandes e Textos Pequenos

Para aumentar determinado trecho dentro de um parágrafo é necessário editar a formatação da letra.
O entendimento comunitário é que não há uma resposta semântica para o aumento no tamanho da fonte.

Para diminuir a fonte, o entendimento é outro.
Entende-se que há casos onde fontes menores denotem algum significado

Texto Deletado

Podemos indicar em paragrafo que determinado trecho será deletado e deve ser desconsiderado com a tag <del>

Texto Inserido

Podemos também indicar que um trecho foi recém inserido com a tag <ins>.

É possível sublinhar determinado trecho de um parágrafo utilizando a tag <u>, porém ela não tem desempenho semântico

Texto Sobrescrito

São textos com elementos elevados, como potência em equação matemática, exemplo: x20 + 3.
A técnica para identificar esses elementos é a tag <sup>

Texto Subescrito

São textos com elementos posicionados abaixo da linha padrão, como por exemplo: H2O.
A tag utilizada para tal função é a <sub>.
