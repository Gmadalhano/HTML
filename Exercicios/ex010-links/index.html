Utilizando Links Externos

Links são chamados de ancoras e levam a tag <a>.

Obs: Sempre que o link direcionar para um outro site, recomenda-se que o link direcione a abertura do site em uma outra aba do navegador, utilizando os parâmetros "target='_blank'", "rel='External'" de forma que não sobreponha (comportamento natural) a pagina atual de navegação. Quando o link encaminhar para outras áreas do mesmo site, não há problmas quanto a sobreposição.

Exemplo:

Você pode acessar ao repositório publico do professor Gustavo Guanabara

Ele também tem um canal no youtube, que vale a pena conferir.

Utilizando links Internos

Essa é a primeira página do nosso site, para continuar navegando clique aqui

Acesse também a nossa página de notícias.

A navegação interna leva o parâmetro "rel='next'"

Existe um parâmetro para o "target" quando a nevegação é interna. Utiliza-se o "_self" (target='_self')
