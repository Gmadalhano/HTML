Estilos "Locais"

Estilos locais são atributos de CSS declarados no cabeçalho do site. Através do comando style indica-se os coletores (tags) e pra cada coletor indicado a sua configuração de estilo.

São ideais para situações de página unica, ou duas, os estilos tem que ser declarados em todos os cabeçalhos.
