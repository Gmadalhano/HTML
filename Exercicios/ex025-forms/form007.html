Exemplo de Formulário

Esse formulário foi feito utilizando a tag < select >, nela não é permitido que o usuário insira dados, como no caso de estados do Brasil, as opções foram agrupadas através da tag < optgroup >

Esse formulário foi feito utilizando a tag < input > de type:text. Nela é possível inserir sugestões de preenchimento, com a tag < datalist > mas o campo é aberto para digitação, caso a opção desejada não se encontre dentro das opções.


Para a caixa de mensagem existe a tag "textarea" que pede a discriminação da quantidade de colunas e linhas. Para fazer a relação com o título da caixa deve utilizar o mesmo procedimento da tag Label
