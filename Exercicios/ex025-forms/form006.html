Testando Formulários

Então vimos os seguintes inputs (types):

  • text
  • password
  • range
  • color
  • submit
  • reset
  • month
  • number
  • date
  • time
  • email
  • tel
  • radio
  • checkbox

Para relacionar a elemento <input> com sua descrição no formulário, a descrição deve estar envelopada dentro da tag <label>, que vai solicitar o "id" do elemento input para fazer a relação.
