Trabalhando com Listas

Existem 3 tipos de listas, listas ordenadas <ol> (ordered list), listas não ordenadas <ul> (unordered list), e as listas mistas.

Cada lista tem um parametro "type", que indica o tipo de lista, númerica, alfabética, números romanos, indices no caso da ordenada, e disc, circle e square no caso das não ordenadas, indicando o simbolo que acompanhará a lista Os itens lista são indicados com a tag <li> (list itens).

Listas Mistas

São listas compostas por outras listas, como indices de livros por exemplo.

  1. O que é HTML5?
    1. Quando surgiu.
    2. Quem inventou.
    3. Como se difundiu.
  2. O que é CSS3
    1. Como aplicar
    2. Conceitos e técnicas
  3. A Organização de um site
  4. Interação com o usuário
  5. Conclusão
    • Autor:
    • Ano de lançamento:
    • Edição:

Listas de Definição

São como dicionários, listando termos e suas definições. As listas de definições tem a tag <dl>, os termos são declarados pela tag <dt> e seus respectivos significados com a tag <dd>. Ex:


HTML5
Linguagem de marcação para criação de conteúdo de um site
CSS3
Linguagem de marcação para a criação de desing de um site
JavaScript
Linguagem de programação para a criação de interatividade de um site.
