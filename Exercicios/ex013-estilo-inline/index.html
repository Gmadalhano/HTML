Estilos "In Line"

Estilos "in line" são edições de estilo de uma terminada tag, declarada diretamente na linha da tag.
Utiliza-se o parâmetro style para alterar a cor de fundo (background-color) na tag body, ou color para editar a cor de um título na tag hx. Essa opção de formatação é indicada apenas para alterações pontuais no estilo do conteúdo, não para o desing do site
